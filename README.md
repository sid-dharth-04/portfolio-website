# 🌐 Siddharth's Portfolio Website

A personal portfolio website built with pure HTML and CSS — no frameworks, no dependencies.
Designed with a clean blue and soft color theme to showcase my skills, projects, and contact information.

## 🚀 Features

- Sticky navigation bar with smooth scroll
- Hero section with animated avatar ring
- About Me cards
- Technical & Soft Skills section
- Projects section with GitHub links
- Contact section with email and hobbies
- Fully responsive (mobile + desktop)

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, grid)
- Google Fonts (Inter + Space Grotesk)
- No frameworks or libraries
